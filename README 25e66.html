Intro

There are 2 main parts here, one for loading data, another is the perceptron object.

Part 1: Loading data

Two modes to control the different style of loading data.

mode-1: Generally for Question-3, using label_1 & label_2 to pick the corresponding data from the whole set.

mode-2: Generally for Question-4&5, returning the whole dataset.

Part 2: Perceptron

2 Functions and some arguments.

Main_label: This is the class/label that chosen for positive prep value(+1). It also means other class(es)/label(s) will be negative prep value(-1) automatically.

Trian(): training the perception with specific iterations. It will directly return the training acc after all iterations done.

test_accuracy(): Return the output for testing set.

Start and initialization

  1. Put the data sets files (train.data/test.data) under folder ‘Asset’

    or amend the file path in load_data().

  1. Initializing these arguments in main() and then just running it.

    mode_select: Entering ‘mode-1’ (Q3) or ‘mode-2’ (Q4, Q5)

    label_1, label_2, label_main: Entering ‘class-1’, ’class-2’ or ‘class-3’

    For mode-1. label_1 & label_2 are required.

    For mode-2, these are unimportant.

    coefficient: lambda for l2 reg, entering 0.01, 0.1, 1, 10, 100...

  1. It will directly return the final train & test acc in the last iter.
